# Learning Playwright

Minimal Playwright project with example chapters. Contains basic examples and setup verification scripts.

Project structure
- chapter_01_basics/: basic Playwright examples (`01_basics.js`, `03_verify_setup.js`)
- chapter_02_Java_Concepts/: placeholder directory

Getting started (Node.js)

1. Install dependencies

```bash
npm init -y
npm install -D playwright
npx playwright install
```

2. Run a script

```bash
node chapter_01_basics/01_basics.js
```
